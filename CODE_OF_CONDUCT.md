# 貢獻者行為準則

## 我們的承諾

為了營造開放友善的環境，我們作為貢獻者與維護者承諾：讓參與本專案與社群的每個人，
無論年齡、體型、身心障礙、族群、性別認同與表達、經驗程度、國籍、外貌、種族、宗教、
性傾向為何，都能有免於騷擾的體驗。

## 我們的準則

有助於營造正向環境的行為包括：

- 使用友善包容的語言
- 尊重不同觀點與經驗
- 坦然接受建設性的批評
- 以社群最大利益為念
- 對其他社群成員展現同理

不被接受的行為包括：

- 帶有性意味的言語或影像，以及任何形式的性騷擾
- 挑釁、侮辱／貶損的評論，人身或政治攻擊
- 公開或私下的騷擾
- 未經明確許可，公布他人的私人資訊（如實體或電子地址）
- 其他在專業場合合理判斷為不當的行為

## 技術討論的補充守則

本專案的核心是「可辯護的規則」。技術爭論請對事不對人：

- 反對一條規則時，附**正面／反面 fixture** 或具體反例，而不是主張「感覺不對」。
- 「我不喜歡」不是理由；「這條規則會誤罰這個可溯源的真值」才是。

## 執行

請透過 repository 的 private vulnerability reporting form 回報辱罵、騷擾或其他不被接受的行為，
並在標題前加上 `[conduct]`。不要在公開 issue 放入敏感內容。所有申訴都會被審閱與調查，
並做出適當回應；維護者有義務對回報者的身分保密。
